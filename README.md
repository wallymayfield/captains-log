# Captain's Log

A local-first markdown writing tool for blog posts and essays, themed
entirely after LCARS — the Library Computer Access and Retrieval System
from Star Trek: TNG.

> **Status:** Step 1 of the build. Chrome only. Editor, file IO, sounds,
> and installers come in subsequent steps.

## Develop

Requires Node 20+ and Rust (stable). Install Rust from <https://rustup.rs>
if you don't have it.

```sh
npm install
npm run tauri:dev
```

The full README, screenshots, and per-OS build instructions are written
in step 9.
